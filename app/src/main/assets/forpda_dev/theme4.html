В этой теме можно потренироваться в загрузке файлов и картинок.

Нажимаем , ниже поля для ввода текста, есть поле "Прикрепление файлов". Далее просто выбираем файл, нажав "Обзор...", жмём "Загрузить".
При нажатии справа есть ссылка на прикрепление файлов, нажимаем ее, появляется форма для загрузки файлов. Можно загрузить несколько файлов.
Т.е. теперь каждый файл привязан к конкретному сообщению. "Библиотеки картинок", которая была на старом движке, нет.
Максимальный размер прикрепляемого файла - 128 Мб.
